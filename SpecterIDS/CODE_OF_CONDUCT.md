# Code of Conduct

SpecterIDS is a defensive security learning project. Be respectful, constructive and precise.

## Expected Behavior

- Help others learn safely.
- Keep discussion focused on defensive use.
- Document risks and limitations honestly.
- Avoid harassment, gatekeeping or unsafe guidance.

## Unacceptable Behavior

- Requests or contributions that add offensive abuse capability.
- Harassment or personal attacks.
- Sharing secrets, credentials or private traffic captures.

Maintainers may remove content that conflicts with the project’s defensive scope.
